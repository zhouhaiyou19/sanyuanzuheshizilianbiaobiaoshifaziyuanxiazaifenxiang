三元组表示法是一种用于稀疏矩阵存储的方法。该方法通过两个结构体来定义：

1. **非零元素结构体**：用于保存每个非零元素的位置（行号和列号）及其值。
2. **矩阵信息结构体**：用于定义整个矩阵的基本信息，包括矩阵的行数、列数、非零元素的个数等。

通过这种方式，三元组表示法能够有效地存储和处理稀疏矩阵，减少不必要的空间浪费。
